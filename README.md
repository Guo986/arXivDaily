# arXivDaily

自动获取 arXiv 的最新文章，通过关键词和 arXiv API

## 说明

直接运行 run.py

即可根据 keywords 获取到从今天凌晨开始，往前推移 pastDays 天的文章，文章数量上限为 total_results。结果会直接输出成 json 格式。
